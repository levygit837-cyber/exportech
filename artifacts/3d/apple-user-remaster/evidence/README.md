# Comparações A/B do remaster modelo de referência

Cada arquivo em `comparison/` mostra o original à esquerda e o remasterizado à direita, com a mesma câmera, HDRI, exposição, luz e resolução. A aprovação visual deve ser humana porque o ativo de origem está marcado como restrição de uso.

Ordem das pranchas:

1. frente em três quartos;
2. frente;
3. lateral direita;
4. traseira;
5. macro das câmeras;
6. lateral esquerda;
7. macro dos botões.

Revisar especialmente cor do alumínio, continuidade dos highlights, vidro frontal e traseiro, anéis, lentes, flash, LiDAR, botões, costuras UV, z-fighting, serrilhado e esmagamento dos pretos. O remaster só deve substituir o material original de um componente quando a melhora for clara em todos os ângulos relevantes.

Nenhuma prancha foi enviada a serviço de visão ou geração. Os PNGs foram renderizados e combinados localmente por operações determinísticas.
